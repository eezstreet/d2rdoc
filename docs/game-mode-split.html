Game Mode Split

Starting in version 3.0, the concept of splitting the data for different game modes was introduced. This means that there are multiple versions of certain data files, with each version being used for a specific game mode. There is no inheritance or anything between the two versions. The split is set up like so:

  • Classic\Expansion modes use files in the /excel/base/ folder
  • Reign of the Warlock mode uses files in the /excel/ folder

There are exceptions to this. Some files are shared across all game modes and thus do not need to be split. For these cases, the files in the /excel/ folder are always used. These files are:

  • sounds.txt
  • soundenviron.txt

It is important to be mindful of a few things due to this change:

  • Indices should match across game modes. For example if in Classic/Expansion mode, a unique item's index in uniqueitems.txt needs to be the same index in the Reign of the Warlock mode's uniqueitems.txt. If the index corresponds to different items, then there will be issues if a character is upgraded from Classic/Expanion to Reign of the Warlock.
  • When making changes, consider what game modes they should go in. For example, if a change is made to the Reign of the Warlock game mode, ask if it should also be made in Classic/Expansion.
